# Metamorphosis

[***Die Verwandlung***](https://en.wikipedia.org/wiki/The_Metamorphosis) is an
experimental .NET Client for [Apache Kafka](http://kafka.apache.org/).

# Requirements

Metamorphosis currently is currently build with .NET 4.6.1
